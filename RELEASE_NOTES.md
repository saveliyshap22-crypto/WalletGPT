# WalletGPT 0.1.0 beta

Первая Android-версия WalletGPT.

- подключение к Tonkeeper через TON Connect v2;
- просмотр публичного адреса, TON и Jetton-балансов;
- отдельный Sandbox с любыми виртуальными валютами, суммами и ценами;
- локальное хранение Sandbox на устройстве;
- тёмный адаптивный интерфейс;
- приложение не запрашивает и не хранит seed-фразу;
- в этой версии нет отправки транзакций.

> APK подписан тестовым ключом GitHub Actions и предназначен для ознакомительной beta-версии. WalletGPT не является продуктом Tonkeeper.
